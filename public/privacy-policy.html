Komitmen Privasi: Kami menghargai privasi Anda. Dokumen ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda saat menggunakan layanan Pasè FX Trader Hub.

1. Informasi yang Kami Kumpulkan

1.1 Informasi yang Anda Berikan

  • Nama dan kontak WhatsApp (jika Anda menghubungi kami)
  • Pertanyaan atau feedback yang Anda kirimkan
  • Data trading journal (tersimpan di browser Anda, bukan di server kami)

1.2 Informasi Otomatis

  • Alamat IP (untuk keamanan dan analitik)
  • Browser dan sistem operasi yang Anda gunakan
  • Halaman yang Anda kunjungi dan waktu akses
  • Referrer (dari mana Anda mengakses website)

1.3 Data yang TIDAK Kami Kumpulkan

  • Tidak ada registrasi akun yang diperlukan
  • Tidak ada data pembayaran/kartu kredit
  • Tidak ada data pribadi sensitif (KTP, NPWP, dll)
  • Data trading journal hanya tersimpan di browser Anda (localStorage)

2. Bagaimana Kami Menggunakan Informasi

  • Menyediakan layanan: Menampilkan konten dan tools yang relevan
  • Perbaikan layanan: Menganalisis penggunaan untuk meningkatkan website
  • Komunikasi: Menjawab pertanyaan Anda via WhatsApp/Email
  • Keamanan: Mendeteksi dan mencegah aktivitas mencurigakan
  • Analitik: Memahami traffic dan perilaku pengguna (tanpa identifikasi personal)

3. Penyimpanan Data Trading Journal

Penting: Data trading journal Anda disimpan secara LOKAL di browser Anda menggunakan teknologi localStorage. Ini berarti:
  • Data TIDAK dikirim ke server kami
  • Data TIDAK dapat kami akses
  • Data akan hilang jika Anda menghapus browser data
  • Data tidak tersinkronisasi antar device

4. Cookie dan Teknologi Pelacakan

4.1 Cookie yang Kami Gunakan

  • Cookie Esensial: Diperlukan untuk fungsi dasar website
  • Cookie Analitik: Membantu kami memahami penggunaan website (Google Analytics/Plausible)
  • Cookie Preferensi: Menyimpan pengaturan Anda (misal: tema)

4.2 Mengelola Cookie

Anda dapat mengontrol cookie melalui pengaturan browser Anda. Namun, menonaktifkan cookie mungkin mempengaruhi fungsionalitas website.

5. Berbagi Informasi

Kami TIDAK menjual data pribadi Anda. Kami hanya membagikan informasi dalam keadaan berikut:

  • Dengan persetujuan Anda: Jika Anda secara eksplisit menyetujui
  • Kewajiban hukum: Jika diwajibkan oleh hukum atau proses pengadilan
  • Keamanan: Untuk melindungi hak, properti, atau keselamatan kami dan pengguna lain
  • Penyedia layanan: Hanya dengan pihak ketiga tepercaya yang membantu operasional (hosting, analitik)

6. Keamanan Data

Kami menerapkan langkah-langkah keamanan yang wajar untuk melindungi informasi Anda:

  • Website menggunakan HTTPS (enkripsi SSL/TLS)
  • Tidak menyimpan data sensitif di server
  • Akses terbatas ke data analitik
  • Regular security updates
Catatan: Tidak ada metode transmisi data melalui internet yang 100% aman. Kami berusaha terbaik tetapi tidak dapat menjamin keamanan absolut.

  • Akses: Meminta salinan data pribadi Anda

  • Penghapusan: Meminta penghapusan data Anda ("right to be forgotten")
  • Batasi penggunaan: Membatasi bagaimana kami menggunakan data Anda
  • Keberatan: Menolak penggunaan data untuk tujuan tertentu
